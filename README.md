Follow these steps to set up and run the project locally.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (version 18+ is recommended).

### Installation

1.  **Install dependencies:**
    Open your terminal in the project folder and run:
    ```bash
    npm install
    ```

2.  **Start the development server:**
    ```bash
    npm run dev
    ```

3.  **Open in browser:**
    Click the link shown in the terminal (usually `http://localhost:5173`) to view the project.

To create an optimized build for deployment (generates a `dist` folder):
